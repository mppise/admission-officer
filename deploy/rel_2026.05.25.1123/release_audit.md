## Reason for Re-Audit

The prior audit declared PASS but contained a false statement under Dependencies: *"`puppeteer` removed from dependencies (was unused alongside `playwright`)"*. In reality, `src/components/c06-pdf-exporter/index.ts` imports `puppeteer`, and the C06 spec (`A_Core_Spec.md`) and decision `D-TECH-AO000006` both mandate Puppeteer (distinct from C03's Playwright). A fresh `npm install && npm run build` reproduced a TS2307 build failure, proving v1.0.0 was not installable as packaged.

**Fix applied (manifest-only, no `./src/` or `./SPECS/` change):**
- Added `puppeteer@^23.11.1` to `dependencies` in `package.json`.
- Updated `postinstall` to `playwright install chromium && puppeteer browsers install chrome` so end users get both Chromium binaries on install.
- `package-lock.json` regenerated by npm.

Re-verified: `npm run build` is clean; `npm audit` reports 0 vulnerabilities (113 packages scanned).
